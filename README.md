# Infinitechain Nodejs Demo
## How to Use
### 1. Install & config env
```
$ npm install
```
```
let env = {
    nodeUrl: 'YOUR_NODE_DOMAIN',
    web3Url: 'YOUR_WEB3_DOMAIN',
    signerKey: 'YOUR_ECC_PRIVATE_KEY',
    cipherKey: 'YOUR_RSA_PRIVATE_KEY'
};
```

### 2. Start `server.js`
```
$ npm start
```

### 3. Start `client.js`
```
$ node client.js
```
